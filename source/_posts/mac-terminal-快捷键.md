常用快捷键

| 快捷键 | 功能                 |
| ------ | -------------------- |
| ⌃ ->   | 光标向前移动一个单词 |
| ⌃  <-  | 光标向后移动一个单词 |
| ⌃ A    | 光标移动到行尾       |
| ⌃ E    | 光标移动到行首       |
| ⌃ U    | 清除当前行           |
| ⌃ D    | 退出当前shell        |
| ⌃ C    | 中断信号             |
| ⌘ K    | 清屏                 |
