| 功能   | 源代码                       | 快捷键   |
| ------ | ------------------------------- | -------- |
| 标题 | `# 文字`                        | `⌘ 数字` |
| **加粗** | `**文字**`      | `⌘B`     |
| *斜体* | `*文字*`                   | `⌘I`     |
| <u>下划线</u> | `<u>文字</u>` | `⌘U`     |
| [超链接](https://www.baidu.com) | `[文本](url)` | `⌘K`     |
| 有序列表 | `1.   ` | `⌥⌘O` |
| 无序列表 | `* ` | `⌥⌘U` |
| 代码块 | `  ```  代码块 ```  ` | `⌥⌘C` |
| `代码行` | ` `` ` | ` ⌃` ` |
|   引用     |             >                     |     `⌥⌘Q`     |
|   公式块     |             ` $$ 公式 $$ `                     |     `⌥⌘B`     |
|  插入图片      |         `  ![](图片地址) `          |    `⌃⌘I`      |
|  注释      |        ` <!--注释内容--> `       |    `⌃-`      |
| 列表缩进-增加缩进 |  | `⌘【` |
| 列表缩进-减少缩进 |                                 | `⌘】` |
| 表格 |  | `⌥⌘T` |
|  切换源码模式      |                                 |    `⌘L`      |
